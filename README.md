# SK Gas H2 app

### Featuring

- iOS
- Android
- Web
- Typescript
- React-Navigation
- Jest
- Eslint


### Install dependencies

```sh
brew install node
brew install watchman
sudo gem install cocoapods
npm i -g yarn
```

### Install code

```sh
yarn install
cd ios
pod install
```

### Set Husky
```sh
yarn remove husky
yarn add -D husky@4.0.0
yarn add -D husky
```

### Run

```sh
yarn ios
```

```sh
yarn android
```

```sh
yarn web
```
### SourceTree with Husky
> install sourcetree cli and...
```sh
stree
```

### Test

```sh
yarn test
```
